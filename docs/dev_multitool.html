Генерируйте хеши MD5, SHA-1, SHA-256, SHA-512. Проверка целостности файлов и паролей.

MD5 - 128-битный хеш (устаревший, не для паролей)
SHA-1 - 160-битный хеш (не рекомендуется для безопасности)
SHA-256 - 256-битный хеш (рекомендуется)
SHA-512 - 512-битный хеш (самый безопасный)
