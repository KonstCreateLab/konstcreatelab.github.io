Условия пользования

Правила использования инструментов и сервисов ItzKonst

Общие положения

Настоящие Условия пользования (далее — Условия) регулируют использование веб-сайтов и инструментов, доступных на домене konstcreatelab.github.io под брендом ItzKonst.

Используя любой из представленных инструментов или сервисов, вы соглашаетесь с настоящими Условиями, а также с нашей Политикой конфиденциальности.

Все инструменты ItzKonst работают полностью в браузере и не требуют регистрации или создания учетной записи.

Доступные инструменты и их особенности

На сайте ItzKonst представлены следующие инструменты:

Игра "Змейка"

Классическая браузерная игра. Все данные (счет, рекорд) хранятся локально в вашем браузере.

Колесо фортуны

Инструмент для принятия случайных решений. Настройки сохраняются локально в вашем браузере.

Эмулятор терминала Linux

Образовательный инструмент для изучения команд терминала. Работает полностью в браузере, без доступа к реальной системе.

DevToolkit

Набор инструментов для разработчиков (JSON форматер, кодировщик, regex тестер и другие). Все вычисления выполняются локально.

Условия использования инструментов

При использовании инструментов ItzKonst вы соглашаетесь:

  1. Использовать инструменты только в законных целях
  2. Не пытаться взломать, изменить или нарушить работу инструментов
  3. Не использовать инструменты для создания вредоносного контента
  4. Не перегружать сервисы чрезмерным количеством запросов
  5. Не использовать автоматизированные скрипты для взаимодействия с инструментами без явного разрешения

Важно: Эмулятор терминала Linux является учебным инструментом и не предоставляет доступ к реальной системе. Не вводите реальные пароли или конфиденциальные данные.

Ограничения ответственности

Инструменты ItzKonst предоставляются "как есть" без каких-либо гарантий:

  • Мы не гарантируем бесперебойную работу инструментов
  • Мы не несем ответственности за любые потери данных, возникшие при использовании инструментов
  • Мы не несем ответственности за решения, принятые на основе работы инструментов (например, результаты "Колеса фортуны")
  • Инструменты DevToolkit предназначены для вспомогательных целей и не заменяют профессиональное программное обеспечение

Для разработчиков: Исходный код большинства инструментов открыт и доступен на GitHub. Вы можете использовать его в образовательных целях или для создания собственных проектов с указанием авторства.

Запрещенное использование

Запрещается использовать инструменты ItzKonst для:

  • Создания или распространения вредоносного программного обеспечения
  • Обхода систем безопасности или защиты
  • Незаконной деятельности любого рода
  • Нарушения прав интеллектуальной собственности
  • Рассылки спама или нежелательных сообщений
  • Сбора личных данных без согласия пользователей

Возрастные ограничения

Большинство инструментов ItzKonst не имеют возрастных ограничений и подходят для пользователей всех возрастов. Однако:

  • Игра "Змейка" и "Колесо фортуны" подходят для всех возрастов
  • Эмулятор терминала и DevToolkit могут быть более интересны пользователям от 13 лет и старше
  • Детям до 13 лет рекомендуется использовать инструменты под присмотром взрослых

Технические требования

Для корректной работы инструментов ItzKonst рекомендуется:

  • Современный веб-браузер (Chrome 80+, Firefox 75+, Safari 13+, Edge 80+)
  • Включенный JavaScript в браузере
  • Доступ к интернету для первоначальной загрузки инструментов
  • Некоторые инструменты могут работать оффлайн после загрузки
  • Минимум 100 МБ свободного места для локального хранения данных

Все инструменты, дизайн и контент на сайте ItzKonst являются интеллектуальной собственностью владельца сайта, если не указано иное.

Вы можете:

  • Использовать инструменты для личных и образовательных целей

  • Изучать исходный код для обучения
